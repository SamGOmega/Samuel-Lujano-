¿Qué es GitHub?

Introducción

GitHub es una plataforma de desarrollo colaborativo para proyectos de software. Se utiliza principalmente para el control de versiones y la gestión de código fuente mediante Git. GitHub permite a los desarrolladores trabajar en proyectos de forma conjunta, gestionar versiones de su código, realizar revisiones y colaborar en equipo.

Características

GitHub ofrece una serie de características clave que facilitan el desarrollo de software en equipo:

  • Control de versiones mediante Git.
  • Herramientas para la colaboración y gestión de proyectos.
  • Revisión y discusión de código mediante "Pull Requests".
  • Automatización de tareas mediante GitHub Actions.
  • Almacenamiento de proyectos en repositorios públicos o privados.

Fundadores de GitHub

GitHub fue fundada en 2008 por los siguientes desarrolladores:

  • Tom Preston-Werner: Desarrollador y emprendedor, fue una de las principales mentes detrás de la creación de GitHub y su diseño.
  • Chris Wanstrath: Ingeniero de software que cofundó GitHub y ayudó a llevarlo a su estado actual como una de las principales plataformas de desarrollo de software.
  • PJ Hyett: Otro de los cofundadores que contribuyó a la creación de la infraestructura técnica y las funcionalidades clave de la plataforma.
  • Scott Chacon: Aunque se unió después de los fundadores iniciales, Scott fue fundamental en el desarrollo del sistema de control de versiones Git, que es el corazón de GitHub.

Adquisición por Microsoft

En 2018, Microsoft adquirió GitHub por 7.5 mil millones de dólares, lo que consolidó aún más su papel como una de las herramientas esenciales en el desarrollo de software en todo el mundo.

Conclusión

GitHub es hoy en día una de las plataformas más utilizadas por desarrolladores y empresas para gestionar proyectos de software. Su capacidad para facilitar la colaboración y la gestión de código lo ha convertido en una herramienta imprescindible en el desarrollo de software moderno.
